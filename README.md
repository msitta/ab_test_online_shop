# ab_test_online_shop
Análise de estratégias para e-commerce usando Python. Priorização de hipóteses (ICE/RICE) e avaliação rigorosa de Teste A/B. Inclui análise de métricas acumuladas (receita, conversão), limpeza de dados e testes de significância estatística para validar decisões de negócio e aumento de receita.
